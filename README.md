# D-Day 카운터 📅

중요한 날짜를 D-Day로 관리하는 미니멀 PWA 앱입니다.

## 기능

- 📅 **이벤트 관리** - 생일, 기념일, 시험, 여행 등 다양한 이벤트
- ⏰ **D-Day 계산** - 남은 날짜 자동 계산 (D-N, D-Day, D+N)
- 🎨 **카테고리별 색상** - 6가지 카테고리 (생일, 기념일, 시험, 여행, 업무, 기타)
- 🔄 **정렬 기능** - 가까운 순, 먼 순, 최근 추가 순
- 📊 **통계** - 전체 이벤트, 다가오는 이벤트, 지난 이벤트
- 🌓 **다크/라이트 모드** - 눈에 편안한 테마
- 📱 **PWA 지원** - 모바일 홈 화면에 설치 가능
- 💾 **자동 저장** - LocalStorage로 데이터 보존

## 기술 스택

- HTML5, CSS3, Vanilla JavaScript
- Progressive Web App (PWA)
- LocalStorage for data persistence
- Responsive Design

## 로컬 실행

```bash
# 간단한 HTTP 서버 실행
python -m http.server 8000

# 브라우저에서 접속
http://localhost:8000
```
